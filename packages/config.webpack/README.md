# @zero-scripts/config.webpack

Base configuration for Webpack

WARNING: This package can't need to install manually. It's used by presets and extensions

## Options

| Option    | Type      | Default |
| --------- | --------- | ------- |
| sourceMap | _Boolean_ | `true`  |

## Passing options

```
{
  "zero-scripts": {
    "@zero-scripts/config.webpack": {
      "sourceMap": false
    }
  }
}
```
